# SoundJao

SoundJao permite baixar músicas e playlists diretamente do SoundCloud, adicionando metadados e capas automaticamente.

## Criado por JaoPred0

## Baixando o Repositório

Clone o repositório do GitHub executando:

```sh
git clone https://github.com/JaoPred0/SoundJao.git
```

Entre na pasta do projeto:

```sh
cd SoundJao
```

## Requisitos

Antes de começar, instale os pacotes necessários executando:

```sh
pip install -r requirements.txt
```

## Instalação e Uso

### Windows

1. Certifique-se de que o Python está instalado.
2. Instale os pacotes necessários com:

   ```sh
   pip install -r requirements.txt
   ```

3. Execute o script no terminal com:

   ```sh
   python run.py
   ```

### Termux (Android)

1. Instale o Termux na Play Store.
2. Atualize os pacotes:

   ```sh
   pkg update && pkg upgrade
   ```

3. Instale Python:

   ```sh
   pkg install python
   ```

4. Instale os pacotes necessários:

   ```sh
   pip install -r requirements.txt
   ```

5. Execute o script:

   ```sh
   python run.py
   ```

### Linux (Ubuntu/Debian)

1. Instale Python e pip:

   ```sh
   sudo apt update && sudo apt install python3 python3-pip
   ```

2. Instale os pacotes necessários:

   ```sh
   pip install -r requirements.txt
   ```

3. Execute o script:

   ```sh
   python3 run.py
   ```

### MacOS

1. Instale o Homebrew (se não tiver):

   ```sh
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. Instale Python:

   ```sh
   brew install python
   ```

3. Instale os pacotes necessários:

   ```sh
   pip install -r requirements.txt
   ```

4. Execute o script:

   ```sh
   python3 run.py
   ```

## Funcionalidades

- Baixa músicas e playlists do SoundCloud.
- Adiciona automaticamente metadados como título, artista e capa ao MP3.
- Interface interativa no terminal com opções de escolha.
