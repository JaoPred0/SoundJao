# SoundJao

SoundJao permite baixar músicas e playlists diretamente do SoundCloud, adicionando metadados e capas automaticamente.

## Criado por JaoPred0

---

## 🚀 Novidades da Versão 3.3

- **Melhoria na velocidade** para Windows, macOS e Linux com um executável otimizado.
- **Maior qualidade de download** para Termux, melhorando a extração de áudio diretamente pelo terminal do celular.
- **Execução simplificada**: No PC, basta rodar `main.py` e, no Termux, `termux.py`.

---

## 📥 Baixando o Repositório

Clone o repositório do GitHub executando:

```sh
git clone https://github.com/JaoPred0/SoundJao.git
```

Entre na pasta do projeto:

```sh
cd SoundJao
```

---

## 📦 Instalação e Uso

### 🖥️ Windows, macOS e Linux

1. Certifique-se de ter o Python instalado.
2. Instale as dependências necessárias:
   ```sh
   pip install -r requirements.txt
   ```
3. Execute o script:
   ```sh
   python main.py
   ```

📌 **Nota**: Em sistemas Linux e macOS, pode ser necessário usar `python3` ao invés de `python`.

### 📱 Termux (Android)

1. Instale o Termux no FDroid.
2. Rode em uma só linha:
   ```sh
   pkg update && pkg upgrade && pkg install git python && git clone https://github.com/JaoPred0/SoundJao.git && cd SoundJao && pip install -r requirements.txt && chmod +x ~/SoundJao/atualizar.sh && chmod +x ~/verifica_atualizacao.sh && python termux.py
   ```

## 🔥 Funcionalidades

✅ Baixa músicas e playlists do SoundCloud com alta qualidade.

✅ Versão otimizada para Termux, melhorando a qualidade do download pelo celular.
