Sobre o Projeto 🎧

Este projeto foi criado com o objetivo de tornar o download de músicas e playlists do SoundCloud algo simples, eficiente e com alta qualidade. Ele é voltado para pessoas que desejam baixar faixas com suas capas originais e qualidade máxima de áudio, sem complicações técnicas.

🚀 Funcionalidades Principais

  • ✔️ Download de músicas individuais e playlists do SoundCloud
  • ✔️ Áudio em qualidade máxima (.mp3)
  • ✔️ Inclusão automática da capa da música (quando disponível)
  • ✔️ Interface responsiva com Bootstrap
  • ✔️ Animações suaves com Animate.css
  • ✔️ Processamento via AJAX com feedback visual de carregamento

⚙️ Tecnologias Utilizadas

O projeto foi desenvolvido em Python, usando o micro-framework Flask para o back-end e controle das rotas. O sistema de download é alimentado pela poderosa ferramenta yt-dlp, que garante suporte atualizado ao SoundCloud.

  • 🔹 Flask (back-end)
  • 🔹 yt-dlp (download de áudio)
  • 🔹 Bootstrap 5 (estilo responsivo)
  • 🔹 Animate.css (animações)
  • 🔹 JavaScript (requisições assíncronas)

📦 Estrutura do Projeto

A aplicação está organizada de forma modular para facilitar manutenção e escalabilidade:

  • app.py: Ponto de entrada da aplicação Flask
  • routes/: Contém rotas separadas por responsabilidade (main e download)
  • templates/: Contém os arquivos HTML com o layout e conteúdo das páginas
  • static/: Estilos, scripts e animações
  • downloads/: Pasta onde os arquivos baixados são temporariamente salvos

📈 Futuro do Projeto

Este é apenas o começo. Diversas melhorias estão sendo planejadas para as próximas versões:

  • 🔜 Suporte para outras plataformas como YouTube e Spotify
  • 🔜 Editor de metadados da faixa (nome, artista, capa personalizada)
  • 🔜 Histórico de downloads realizados
  • 🔜 Exibição de progresso em tempo real
  • 🔜 Suporte multi-idioma
  • 🔜 Deploy para nuvem (Render, Heroku ou Vercel)

💡 Filosofia

O projeto foi feito com foco em liberdade do usuário, simplicidade e código aberto. A ideia é democratizar o acesso a conteúdos de música offline, sempre respeitando os direitos autorais e incentivando o uso justo.
